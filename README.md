# OA_SSM
OA办公管理系统

## 功能
1. 部门信息管理
2. 员工信息管理
3. 报销单处理
4. 请假单处理

## 主要技术
1. Mysql
3. Spring MVC
4. Spring IOC
5. Mybatis + Spring 整合 
6. 声明式事务
7. Spring 标签库
8. Spring 拦截器
